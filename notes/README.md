# notes/

Ephemera. Dated working notes — sprint logs, RCAs, bench write-ups, design
proposals, slides — that earned a place in version control but do not belong
next to canonical reference. These files are not canon and are not bound by
[`docs/DOCS_RUBRIC.md`](../docs/DOCS_RUBRIC.md). They are kept so the trail
of decisions and measurements survives, not as documentation a reader should
expect to be current.
